✅ Lista de Tarefas — React
Aplicação simples de gerenciamento de tarefas feita com React e hooks.
🚀 Funcionalidades

Adicionar novas tarefas
Marcar/desmarcar tarefas como concluídas
Excluir tarefas individualmente
Contador de tarefas totais e concluídas
Campo limpo automaticamente após adicionar

🛠️ Tecnologias

React
useState (hooks)

📁 Estrutura
src/
├── App.jsx
└── components/
    └── Lista.jsx
    
▶️ Como rodar
bash# Instale as dependências
npm install

# Inicie o projeto
npm run dev
